Objective

ถ้าทีม 8–12 คน ต้องเลือกร้านมื้อเย็นหลังเลิกงาน 1–3 ร้านในย่านอารีย์ — ร้านไหนเหมาะที่สุดและเพราะอะไร ตัดสินจากข้อมูลจริง + รีวิวจริงที่ตรวจย้อนกลับได้ ผ่านโมเดลให้คะแนน 100 คะแนน 6 มิติ

Workflow Overview

  1. Data Collection — Apify (Google Maps) + Browse AI (Wongnai) → 72 + 100 ร้าน
  2. Storage — Google Sheets (แยก raw / clean ชัดเจน)
  3. Cleaning — dedup, normalize ราคา→ต่อหัว, กรองรัศมี ≤1.5km จาก BTS อารีย์, cross-check 2 แหล่ง
  4. Scoring + AI Analysis — โมเดล 100 คะแนน 6 มิติ + AI อ่านรีวิวจริงให้คะแนน Group/Uniqueness (sentiment)
  5. Automation — n8n Webhook API คืน Top 3 เป็น JSON (รันจริง)
  6. Output — HTML หน้าเดียว (area selector · 5 scenario · Grounded AI Ask Box)
